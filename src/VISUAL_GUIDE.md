### 1. Glass Morphism

**CSS Implementation:**
```css

backdrop-filter: blur(20px);
border: 1px solid rgba(168, 85, 247, 0.2);
```

**Effect:** Frosted glass with purple tint  
**Used In:** Cards, modals, overlays

### 2. Neon Glow

**CSS Implementation:**
```css
box-shadow: 0 0 20px rgba(168, 85, 247, 0.8);
filter: drop-shadow(0 0 10px rgba(168, 85, 247, 0.5));
```

**Effect:** Soft purple glow around elements  
**Used In:** Icons, buttons, particles

### 3. Gradient Text

**CSS Implementation:**
```css
background: linear-gradient(to right, #a78bfa, #ec4899, #3b82f6);
background-clip: text;
-webkit-background-clip: text;
color: transparent;
```

**Effect:** Multi-color flowing text  
**Used In:** Headings, logo, numbers

### 4. Pulsing Animation

**CSS Implementation:**
```css
@keyframes pulse-glow {
  0%, 100% { opacity: 1; filter: drop-shadow(0 0 20px rgba(168, 85, 247, 0.8)); }
  50% { opacity: 0.6; filter: drop-shadow(0 0 40px rgba(168, 85, 247, 1)); }
}
```

**Effect:** Breathing light effect  
**Used In:** Brain icon, neurons, highlights

- Duration: 0.6s

- Duration: 20s infinite
- Movement: Random circular path
- Used: Background particles

- Duration: 2s infinite
- Effect: Brightness pulse
- Used: Icons, highlights

- Used: Brain icon, orbs

- Duration: 1.5s infinite
